# shipping-logistics-database
SQL database design project for a shipping and logistics management system.

This project is a SQL database design for a shipping and logistics management system.

The database is designed to manage vessels, routes, sailing schedules, ports, containers, customer bookings, cargo tracking, and crew assignments.

## Files Included

- `shipping_logistics_database.sql` – SQL code for the database tables, relationships, and queries.
- `database_schema.drawio` – Editable database schema diagram created using draw.io.

## Key Features

- Manage vessel schedules and service routes
- Track cargo using booking details
- Search available containers for booking
- Manage customer bookings
- Calculate vessel capacity usage
- Organize ports, vessels, containers, and schedules in a relational database structure

## Skills Demonstrated

- SQL
- Database design
- Entity Relationship Diagram design
- Relational database modelling
- Primary and foreign keys
- Table relationships
- Data normalization
